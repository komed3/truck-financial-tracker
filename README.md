# Truck Financial Tracker

Financial tracking application for SCS games (Euro Truck Simulator 2 and American Truck Simulator)

## Installation

Clone the repository:

```bash
git clone https://github.com/komed3/truck-financial-tracker.git
cd truck-financial-tracker
```

Install npm environment:

```bash
npm install
```

Start the app:

```bash
npm start
```

Open the truck financial app on http://localhost:3000
